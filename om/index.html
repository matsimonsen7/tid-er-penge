Om Tid er Penge

Hvad er Tid er Penge?

Tid er Penge er en gratis investeringsberegner, der viser dig hvad dine penge kunne være vokset til, hvis du havde investeret i udvalgte aktier. Formålet er at illustrere kraften i langsigtet investering — og hvad tid gør ved dine penge.

Data og opdateringer

Aktiedata hentes fra Yahoo Finance og opdateres automatisk hver uge via GitHub Actions. Beregningerne er baseret på historiske lukkepriser og inkluderer ikke handelsomkostninger, skat eller inflation.

Affiliate-samarbejde

Tid er Penge indeholder affiliate-links til eToro. Det betyder, at vi kan modtage en kommission hvis du opretter en konto via vores links. Dette påvirker ikke beregningerne eller indholdet på siden. 51% af private CFD-konti taber penge.

Ansvarsfraskrivelse

Indholdet på denne side er kun til uddannelsesformål og udgør ikke investeringsrådgivning. Tidligere afkast er ingen garanti for fremtidige resultater. Investering indebærer risiko, og du kan miste hele eller dele af dit investerede beløb. Søg altid professionel rådgivning før du investerer.
